Navegação em abas As abas trazem dinamismo a aplicação através do uso de javascript

O conceito de abas poderá ser usado para dinamizar e facilitar a navegação entre as informações do mesmo gênero. As abas auxiliam a trazer as informações para o usuário de forma separada, facilitando o entendimento da mesma. Dessa forma o usuário poderá também localizar rapidamente a área que deseja visualizar, sem que para isso preciso utilizar muito o scroll para navegar em uma página complexa.

Usando navegação em abas é necessário o Plugin Javascript de Abas do Bootstrap.
A guideline RNP já provê o conjunto de bibliotecas JavaScript do Bootstrap, sendo assim, todos os componentes já estão integrados.

Você pode ativar a navegação sem escrever qualquer JavaScript simplesmente especificando data-toggle="tab" ou data-toggle="pill" em um elemento. Adicionando as classes nav e nav-tabs para a tab ul serão aplicadas a guia de estilos de tabs do Bootstrap, acrescentando as classes nav e nav-pills serão aplicadas guia de estilos de pills do Bootstrap.

Navs disponíveis no Bootstrap tem marcação compartilhada, começando com a classe base .nav, bem como estados compartilhados. Troque as classes modificadoras para alternar entre cada estilo.

Mais informações em:
http://getbootstrap.com/components/#nav
