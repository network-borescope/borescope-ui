Formulário

Quando possível, os formulários da aplicação devem possuir a classe form-horizontal.

Tipos de formulários

Nossa biblioteca possui 4 tipos diferentes de organização de formulários.

Nome Classe Descrição
Vertical (default) .form-vertical
(não obrigatório)		 O rótulo do campo fica acima do controle, sempre alinhado à esquerda. É o formato padrão de formulários que vão dentro de janelas modais e espaços reduzidos.
Inline .form-inline Rótulo e controle ficam na mesma linha, alinhados à esquerda.
Busca .form-search Formulário que já vem com um padrão estético de um campo de busca
Horizontal .form-horizontal labels com float left alinhados à direita, na mesma linha dos controles.

Onde utilizar cada tipo de formulário?

Formulários na página

Para formulários utilizados diretamente na página, sem utilização de componentes como janelas modais, utilize a classe .form-horizontal. Tipos de Campo

Tipos básicos

Nome Tipo Aparência
Texto type="text"
Checkbox type="checkbox"
Radio type="radio"
Select select
Select List select multiple="multiple"
Arquivo type="file"

Campos especiais

Nome Tipo Aparência Descrição
Date Docs type="text" datepicker mask="date" Ao focar o campo, exibe um calendário para a seleção de uma data para preenchimento do texto.
Máscara Docs type="text" mask="decimal"
Garante que o campo será preenchido com uma formatação específica.
Select2 Docs select class="select2-input" Incrementa ao funcionamento padrão do select um campo de texto para facilitar a busca de uma opção específica.
Select2 Multiplo Docs select class="select2-input" multiple="multiple" Exibe listas de seleção multiplas em formato de select2.
