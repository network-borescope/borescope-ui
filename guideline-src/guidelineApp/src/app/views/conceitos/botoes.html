Botões do calendário

As classes de botões do calendário podem ser adicionadas as tags: button, a e input. Devem sempre estar dentro de uma estrutura com a classe .rnp-calendar-menu.
Sua composição sempre segue o padrão:
btn rnp-calendar-btn
<button class="btn rnp-calendar-btn">Botão do calendário (btn rnp-calendar-btn)</button>

As classes de grupo de botões do calendário deve ser adicionada a tag div. Deve sempre estar dentro de uma estrutura com a classe .rnp-calendar-menu.
Sua composição sempre segue o padrão:
rnp-calendar-btn-group
<div class="rnp-calendar-menu">
<div class="rnp-calendar-btn-group">
<button class="btn rnp-calendar-btn active">Botão 1</button>
<button class="btn rnp-calendar-btn">Botão 2</button>
</div>
</div>

Botões para ações

Estilos de botão pode ser aplicado a qualquer coisa com a classe btn e btn-default. Aplicada. No entanto, normalmente você vai querer aplicá-las as elementos do tipo button .

Compatibilidade

IE9 não corta gradientes de fundo em cantos arredondados, então resolvemos removê-los neste browser. Ainda sobre o IE9, quando desativamos um button , o browser coloca um texto cinza com uma sombra que infelizmente não podemos resolver utilizando os recursos atuais disponíveis.

Desabilitados

Para botões desabilitados, utilize a classe disabled para elementos do tipo button .
