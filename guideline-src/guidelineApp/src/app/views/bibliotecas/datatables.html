jQuery Datatables

O componente de datagrid deve ser utilizado quando a exibição de dados necessitar de filtragem, ordenação e paginação dos dados.
A paginação só deve ser aplicada quando o número total de registros for acima de 10.
Com uma simples chamada é possível um datagrid como o exibido ao lado. Estamos utilizando uma biblioteca muito conhecida chamada DataTables (1.10.7) que é desenvolvida utilizando o jQuery e um add-on chamado DataTables-RowGrouping.
Os inputs que possuírem datepicker deverão ser indicados com o ícone de calendário.

Mais informações em:
https://datatables.net
http://jquery-datatables-row-grouping.googlecode.com/svn/trunk/index.html
