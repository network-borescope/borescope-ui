Textarea Limiter

O componente textarea-limiter é um fallback para usuários do browser Internet Explorer.
Assim como os outros componentes ele é acionado automaticamente quando um textarea é inserido no documento e possuir o atributo maxlength.

Browsers modernos como Chrome, Firefox, etc já possuem esse recurso por padrão.
O componente adiciona um label logo abaixo do textarea, exibindo a quantidade de caracteres restantes para o usuário.
Quando o usuário chega no limite nenhum outro caracter é inserido no componente.

<div class="form-group">
<label for="mensagem_input" class="control-label col-xs-2 col-sm-2 col-md-2 col-lg-2">Mensagem</label>
<div class="col-xs-6 col-sm-6 col-md-6 col-lg-6">
<textarea id="mensagem_input" rows="5" name="mensagem" class="form-control" maxlength="200"></textarea>
</div>
</div>
