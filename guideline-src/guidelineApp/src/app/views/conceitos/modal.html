Modal

Modais devem ser utilizados para exibir opções secundárias em uma tela complexa. O mesmo deve conter sempre um cabeçalho contendo um título que informa o conteúdo do modal.
Mais informações:
http://getbootstrap.com/javascript/#modals
Modal padrão
<div class="modal-dialog">
<div class="modal-content">
<div class="modal-header">
<h4 class="modal-title ng-binding">Adicionar Nome</h4>
</div>
<div class="modal-body">
<div class="modal-wrapper">
<span>Conteúdo do modal</span>
</div>
</div>
<div class="modal-footer">
<button type="button" class="btn btn-secondary-color" data-dismiss="modal">Cancelar</button>
<button type="button" class="btn btn-primary-color">Salvar</button>
</div>
</div>
</div>

Exemplo de Modal
Para executar a modal você precisa informar o atributo [data-toggle="modal"] no botão na qual a ação vai ser realizada e o elemento alvo [data-target="#myModal"], ou utilizar o exemplo em javascript para a execução do mesmo.

<div class="modal fade" id="myModal" tabindex="-1" role="dialog" aria-labelledby="myModalLabel">
<div class="modal-dialog" role="document">
<div class="modal-content">
<div class="modal-header">
<button type="button" class="close" data-dismiss="modal" aria-label="Close"><span aria-hidden="true">×</span></button>
<h4 class="modal-title" id="myModalLabel">Título da Modal</h4>
</div>
<div class="modal-body">
Exemplo de Texto
</div>
<div class="modal-footer">
<button type="button" class="btn btn-secondary-color" data-dismiss="modal">Cancelar</button>
<button type="button" class="btn btn-primary-color">Salvar</button>
</div>
</div>
</div>
</div>

<button type="button" class="btn btn-primary-color btn-lg" data-toggle="modal" data-target="#myModal">
Demonstração de Modal
</button>

$("#myModal").modal();
