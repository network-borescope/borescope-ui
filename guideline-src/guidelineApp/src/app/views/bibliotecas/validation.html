Validação

O plugin jQuery Validation é o plugin utilizado em todos os formulários que necessitam dessa característica.
Mais informações em:
http://docs.jquery.com/Plugins/Validation

Exemplo de formulário de validação

O exemplo abaixo mostra um formulário simples com algumas regras de validação aplicadas. Uma lista com todas as validações disponíveis pode ser encontrada na documentação do plugin.

Formulário de cadastro

$(function() {

$("#form").validate({
rules: {
nome: "required",
facebook: "required",
email: {
required: true,
minlength: 2
},
mensagem: {
required: true,
minlength: 5
}
},
messages: {
nome : "Digite o seu nome",
facebook : "Digite o o seu Facebook",
email: {
required: "Digite um email",
minlength: "Seu email deve conter ao menos 2 caracteres"
},
mensagem: {
required: "Digite sua mensagem",
minlength: "Sua mensagem deve conter ao menos 5 caracteres"
}
}
});

});
