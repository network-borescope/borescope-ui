Datepicker

Os inputs que possuírem datepicker deverão ser indicados com o ícone de calendário. O datepicker utilizado nas aplicações da RNP é o jQuery Datepicker.

O estilo para este componente é aplicado automaticamente.

Mais informações em:
https://jqueryui.com/datepicker

Implementação Básica

$( "#datepicker" ).datepicker();

Alcance de Data

$(function(){
var datepickerOptions = {
defaultDate: "+1w",
changeMonth: true,
changeYear: true
};

$( "#from_input" ).datepicker( $.extend(datepickerOptions, {
onSelect: function( selectedDate ) {
$( "#to_input" ).datepicker( "option", "minDate", selectedDate );
}
}));

$( "#to_input" ).datepicker( $.extend(datepickerOptions, {
onSelect: function( selectedDate ) {
$( "#from_input" ).datepicker( "option", "maxDate", selectedDate );
}
}));
});
