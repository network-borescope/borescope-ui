Máscara

O componente jQuery MeioMask determina o tipo de input de dados que o campo do formulário irá receber. O componente fornece vários tipos de máscara, você pode visualizar os tipos na Wiki do projeto.

Mais informações em:
https://github.com/fabiomcosta/jquery-meiomask

Máscara para dados

<input mask="decimal" />
<input mask="cpnj" />
<input mask="cpf" />
<input mask="phone" />

Máscaras básicas

<input mask="numeric" />
<input mask="alpha" />
<input mask="alphanumeric" />

Notas para o desenvolvedor

O plugin de máscaras esta disponível globalmente na aplicação. Você não precisa se preocupar em chamar o método .setMask() em todos os locais onde necessitar do comportamento de máscara. Você só tem que se preocupar em colocar a máscara correta no atributo mask do campo e pronto! Dê uma olhada no trecho de código ao lado.
