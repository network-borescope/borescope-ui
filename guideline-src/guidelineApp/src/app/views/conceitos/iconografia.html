Iconografia

O ícone é uma representação que deve estar em consonância com a cultura da empresa, ou seja, ele deve ser um facilitador de entendimento. O ícone deve ser colocado junto a um botão para que em algum momento do sistema ele possa aparecer sozinho (com texto em mouse over) e continuar fazendo sentido. O ícone deve estar sempre à esquerda do texto que o acompanha (no caso de botões).

Como usar

A Guideline usa uma tag<i> para todos os ícones, mas eles não têm nenhum caso de classe, apenas um prefixo comum. Para usar, coloque o seguinte código em qualquer lugar:


<i class="glyphicon glyphicon-search"></i>

Há mais de 250 classes para escolher a partir de seus ícones.
Basta adicionar uma tag <i> com a classe certa e estará pronto.
Esse sprite de ícones poderá ser aumentado, com isso aumentará o número de classes que representa os ícones.

Mas atenção! Ao utilizar ao lado de áreas de texto, como em botões ou links, não se esqueça de deixar um espaço após a tag <i> para o espaçamento adequado.

Casos de uso

Os ícones são legais, mas onde então devemos usá-los? Aqui estão algumas idéias:

  • Como recursos visuais para a sua barra lateral de navegação
  • Para uma navegação puramente icon-driven "navegação orientada pelos ícones"
  • Em botões para ajudar a transmitir o significado de uma ação
  • Com links para compartilhar contexto do destino de um usuário

Essencialmente, qualquer lugar que você pode colocar uma tag <i>, você pode colocar um ícone.
